<h1 align="center">Socket Server</h1>

Это Socket Server для этого [проекта](https://github.com/YanderuxTea/MultiForum?tab=readme-ov-file). Чтобы запустить
локально необходимо сделать эти шаги:

1. Создать .env файл с ключами
   - PORT=...
   - NODE_ENV=...
   - KEY_CRYPTO=...
   - NEXT_URL=...
   - DATABASE_URL=...
   - JWT_SECRET=...
2. `pnpm install`
3. `pnpm prisma:push` | `npx prisma db push && prisma generate`
4. `pnpm start`
